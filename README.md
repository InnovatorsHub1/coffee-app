## Installation + Running The Application

Steps: 


1. Clone the repository to your local machine:
   ```sh
   git clone https://github.com/InnovatorsHub1/coffee-app.git
   
2. Open folder:
   ```sh
   cd coffee-app

3. Download Dependencies:
   ```sh
   pnpm i

4. Run Project In Local:
   ```sh
   pnpm run dev
